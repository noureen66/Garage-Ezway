### Garage EZway – Smart Parking Management System

## Project Description

Garage EZway is a smart parking management system designed to address the challenges of parking congestion and inefficient space utilization in urban environments. The system aims to help drivers easily find, reserve, and manage parking spaces in real time, while also providing parking operators with better control and monitoring tools.

This project focuses mainly on frontend development using Flutter, with integration of Firebase services to handle authentication, real-time data, and core system functionalities. The goal is to deliver a smooth, responsive, and user-friendly experience across web and mobile platforms.

The system simulates a complete parking flow starting from user authentication, vehicle management, slot reservation, and parking session tracking, all the way to payment processing and history management.

## Key Features

-User authentication system including login, registration, and password recovery using Firebase Authentication

-Ability to add and manage multiple vehicles within the user profile

-Real-time display and updates of parking slot availability using Firebase Realtime Database and Firestore

-Booking and unbooking of parking slots in an interactive and seamless way

-Start and end parking sessions with live tracking of user activity

-Payment flow integration for handling parking session costs (frontend simulation)

-Navigation interface to guide users to selected parking areas

-History section to track previous bookings and parking sessions

-Problem reporting system implemented using a rule-based chatbot interface

-Responsive design ensuring compatibility across different screen sizes and devices

## Technologies Used

-Flutter (Dart) for frontend development across web and mobile

-Firebase Authentication for secure user login and account management

-Cloud Firestore and Realtime Database for handling real-time parking data

-SharedPreferences for local storage and session persistence

The goal of Garage EZway is to reduce time wasted searching for parking, improve traffic efficiency, and provide a smart, automated parking experience using modern web technologies and real-time cloud systems.

## Getting Started

This project is built using Flutter.

To run the project locally:

Install Flutter SDK
Run flutter pub get to install dependencies
Use an emulator or physical device
Run flutter run
